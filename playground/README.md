# playground/

Disposable scripts and experiments kept across sessions but **never committed**
(everything here except this README is git-ignored).

Use it for: one-off analyses, scratch queries, throwaway prototypes you want to
keep on disk longer than the session scratchpad (`/tmp/.../scratchpad`) but that
don't belong in a real repo.

Promote anything worth keeping into its own project or a `workflows/` script.
